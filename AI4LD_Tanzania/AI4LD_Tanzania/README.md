# CLIMAID

## Climate Intelligence for Adaptation and Loss & Damage

CLIMAID is an AI-powered decision support platform for climate adaptation, drought risk screening, and climate-related Loss and Damage assessment.

## Current MVP Features
- Executive dashboard
- Tanzania district climate-risk map
- AI economic loss prediction
- AI non-economic Loss and Damage prediction
- Vulnerability classification
- Analytics and explainability
- Report generator
- Adaptation planner
- Early warning centre
- Climate finance dashboard

## Transparency
This MVP uses simulated household Loss and Damage data for demonstration. Verified field data should replace simulated records before operational or policy use.
